Common Types

These are the types you will need for most common used cases.

AddressString

  • Base64 encoded string that represents a NEO address. 
    'APyEx5f4Zm4oCHwFWiSTaph1fPBxZacYVR'
  • Also accepts Hash160 strings (hex encoded string prefixed by '0x') when used as a parameter to a to a NEO•ONE function. 
    '0xecc6b20d3ccac1ee9ef109af5a7cdb85706b1df9'

Account

  • Contains asset balances for a given address.
  • Properties:
    • address: AddressString
    • balances: {[asset: string]: BigNumber}

Asset

  • Properties
    • hash: Hash256String
      • ID of the Asset
    • type: AssetType
    • name: string
      • Name of the Asset
    • amount: BigNumber
      • Total possible supply of the Asset
    • available: BigNumber
      • Amount currently available of the Asset
    • precision: number
      • Number of decimal places of the Asset
    • owner: PublicKeyString
      • Owner of the Asset
    • admin: AddressString
      • Admin of the Asset
    • issuer: AddressString
      • Issuer of the Asset
    • expiration: number
      • Block at which the Asset expires
    • frozen: boolean
      • Asset is frozen when true

AssetRegister

  • Information required to register an Asset.
  • Properties:
    • type: AssetType
    • name: string
      • Name of the Asset
    • amount: BigNumber
      • Total possible supply of the Asset
    • precision: number
      • Number of decimal places of the Asset
    • owner: PublicKeyString
      • Owner of the Asset
    • admin: AddressString
      • Admin of the Asset
    • issuer: AddressString
      • Issuer of the Asset

BufferString

  • Hex encoded string that represents a buffer. 
    '02028a99826edc0c97d18e22b6932373d908d323aa7f92656a77ec26e8861699ef'

GetOptions

  • Generic options used for different operations.
  • Properties:
    • timeoutMS (optional): number
      • Time before operation timeout

Hash256String

  • Hex encoded string prefixed by '0x' that represents a NEO 256 bit hash.
  • Examples of Hash256String include Block hashes andTransaction hashes. 
     '0x7f48028c38117ac9e42c8e1f6f06ae027cdbb904eaf1a0bdc30c9d81694e045c'

Input

  • Inputs are a reference to an Output of a Transaction that has been persisted to the blockchain.
  • The sum of the values of the referenced Outputs is the total amount transferred in the Transaction.
  • Properties:
    • hash: Hash256String
      • Hash of the Transaction this input references.
    • index: number
      • Output index within the Transaction this input references.

InputOutput

  • Extension of Input and Output.

NetworkType

  • String values which denote the Network
    • 'main'
      • The NEO Main network
    • 'test'
      • The NEO Test network
    • string
      • Arbitrary string for the name of a private network

Output

  • Outputs represent the destination Address and amount transferred of a given Asset.
  • The sum of the unspent Outputs of an Address represent the total balance of the Address.
  • Properties:
    • asset: Hash256String
      • Hash of the Asset that was transferred.
    • value: BigNumber
      • Amount transferred.
    • address: AddressString
      • Destination Address.

PrivateKeyString

  • Hex encoded string that represents a signature for a message. 
    'ccaab040cc25021c91567b75db4778853441869157b8f6aad960cdcf1069812480027a528ca9b98e2205027de20696f848cf81824eeb7af1d5110870870ceb67'

PublicKeyString

  • WIF string that represents a private key. 
    'L1QqQJnpBwbsPGAuutuzPTac8piqvbR1HRjrY5qHup48TBCBFe4g'
  • Also accepts hex encoded strings when used as a parameter to a NEO•ONE function. 
    '9ab7e154840daca3a2efadaf0df93cd3a5b51768c632f5433f86909d9b994a69'
  • Always a WIF string when returned from a NEO•ONE function.

SignatureString

  • Hex encoded string that represents a signature for a message. 
    'ccaab040cc25021c91567b75db4778853441869157b8f6aad960cdcf1069812480027a528ca9b98e2205027de20696f848cf81824eeb7af1d5110870870ceb67'

Transfer

  • Data required to initiate a transfer of an Asset.
  • Properties:
    • amount: BigNumber
      • Amount to transfer.
    • asset: Hash256String
      • Asset to transfer.
    • to: AddressString
      • Address where the Asset is to be transferred.

UserAccount

  • Information associated with a UserAccount
  • Properties:
    • type: string
    • id: [UserAccountID(#UserAccountID)
    • name: string
    • publicKey: PublicKeyString
    • configurableName: boolean
    • deletable: boolean

UserAccountID

  • ID to identify a UserAccount by address and network.
  • Properties:
    • network: NetworkType
    • address: AddressString


Contract Types

Types related to Smart Contracts.

Action

  • Event
  • Log

Contract

  • Definition of a Contract.
  • Properties:
    • version: number
    • address: AddressString
      • AddressString of this Contract
    • script: BufferString
      • Contract code
    • parameters: ReadonlyArray<ContractParameterType>
      • Expected parameters of this Contract.
    • returnType: ContractParameterType
      • Return type of this Contract.
    • name: string
      • Name of this Contract. For informational purposes only.
    • codeVersion: string
      • Version of this Contract. For informational purposes only.
    • author: string
      • Author of this Contract. For informational purposes only.
    • email: string
      • Email of this Contract. For informational purposes only.
    • description: string
      • Description of this Contract. For informational purposes only.
    • storage: boolean
      • True if this contract can use storage
    • dynamicInvoke: boolean
      • True if this Contract can make dynamic invocation.s
    • payable: boolean
      • True if this Contract accepts first-class assets and/or tokens.

ContractParameterType

  • All possible values of the type field for ContractParameter 
    'Signature' | 'Boolean' | 'Integer' | 'Address' | 'Hash256' | 'Buffer' | 'PublicKey' | 'String' | 'Array' | 'InteropInterface' | 'Void'

ContractRegister

  • Information required to register a Contract.
  • Properties:
    • script: BufferString
      • Contract code
    • parameters: ReadonlyArray<ContractParameterType>
      • Expected parameters of this Contract.
    • returnType: ContractParameterType
      • Return type of this Contract.
    • name: string
      • Name of this Contract. For informational purposes only.
    • codeVersion: string
      • Version of this Contract. For informational purposes only.
    • author: string
      • Author of this Contract. For informational purposes only.
    • email: string
      • Email of this Contract. For informational purposes only.
    • description: string
      • Description of this Contract. For informational purposes only.
    • storage: boolean
      • True if this contract can use storage
    • dynamicInvoke: boolean
      • True if this Contract can make dynamic invocation.s
    • payable: boolean
      • True if this Contract accepts first-class assets and/or tokens.

Event

  • Extension of RawActionBase.
  • Properties:
    • type: 'Event'
    • name: string
    • parameters: EventParameters

EventParameters

  • Parameters associated with an Event
  • Mapping of names to Params

Log

  • Extension of RawActionBase.
  • Properties:
    • type: 'Log'
    • message: string

Param

  • Valid parameter types for a smart contract function.
    • undefined
    • BigNumber
    • BufferString
    • AddressString
    • Hash256String
    • PublicKeyString
    • boolean
    • ParamArray
    • ForwardValue

ParamArray

  • Array type allowed to be passed to a smart contract function.
  • ReadonlyArray<Param>

ParamJSONArray

  • Array type allowed inside a JSON object passed to a smart contract function.
  • Extension of ReadonlyArray<Param>

ParamJSON

  • Valid parameter types inside a JSON object passed to a smart contract function.
    • undefined
    • string
    • BufferString
    • AddressString
    • Hash256String
    • PublicKeyString
    • boolean
    • ParamJSONArray

Return

  • Valid return types for a smart contract function.
    • undefined
    • BigNumber
    • BufferString
    • AddressString
    • Hash256String
    • PublicKeyString
    • boolean
    • ReturnArray
    • ContractParameter

ReturnArray

  • ReadonlyArray<Return>

StorageItem

  • An element in persistent Conract storage.
  • Properties:
    • address: AddressString
      • Contract Address for this StorageItem.
    • key: BufferString
      • Key of this StorageItem.
    • value: BufferString
      • Value of this StorageItem.


Transaction Types

Types related to Transactions.

ClaimTransaction

  • Extension of TransactionBase.
  • Claim GAS for a set of spent Outputs
  • Properties:
    • type: 'ClaimTransaction'
    • claims: ReadonlyArray<Input>

ConfirmedClaimTransaction

  • Extension of ConfirmedTransactionBase and ClaimTransaction.

ConfirmedContractTransaction

  • Extension of ConfirmedTransactionBase and ContractTransaction.

ConfirmedInvocationTransaction

  • Extension of ConfirmedTransactionBase and InvocationTransaction.
  • Properties:
    • invocationData: RawInvocationData

ConfirmedIssueTransaction

  • Extension of ConfirmedTransactionBase and IssueTransaction.

ConfirmedMinerTransaction

  • Extension of ConfirmedTransactionBase and MinerTransaction.

ConfirmedStateTransaction

  • Extension of ConfirmedTransactionBase and StateTransaction.

ConfirmedTransaction

  • ConfirmedMinerTransaction
  • ConfirmedIssueTransaction
  • ConfirmedClaimTransaction
  • ConfirmedContractTransaction
  • ConfirmedStateTransaction
  • ConfirmedInvocationTransaction

ConfirmedTransactionBase

  • Base interface for all confirmed Transactions.
  • Properties:
    • receipt:
      • blockHash: Hash256String
      • blockIndex: number
      • index: number
      • globalIndex: BigNumber

ContractTransaction

  • Extension of TransactionBase
  • Transfer of first-class Assets
  • Properties:
    • type: 'ContractTransaction'

InvocationResult

  • Result of an invocation.
    • InvocationResultError
    • InvocationResultSuccess

InvocationResultError

  • Result of a failed invocation.
  • Properties:
    • state: 'FAULT'
    • gasConsumed: BigNumber
    • gasCost: BigNumber
    • message: string

InvocationResultSuccess

  • Result of a successful invocation.
  • Properties:
    • state: 'HALT'
    • gasConsumed: BigNumber
    • gasCost: BigNumber
    • value: Return Value of the Invocation

InvocationTransaction

  • Extension of TransactionBase
  • Runs a script in the NEO-VM.
  • Properties:
    • type: 'InvocationTransaction'
    • script: BufferString
    • gas: BigNumber

InvokeClaimTransactionOptions

  • Extra options available for a ClaimTransaction.
  • Extension of TransactionOptions.
  • Properties (optional):
    • claimAll: boolean
      • True when the claim should claim all unclaimed GAS.

InvokeReceipt

  • Extension of TransactionReceipt for invocations.
  • Properties:
    • result: InvocationResult
    • events: ReadonlyArray<Event>
    • logs: ReadonlyArray<Log>

IssueTransaction

  • Extension of TransactionBase
  • Issues new currency of a first-class Asset.
  • Properties:
    • type: 'IssueTransaction'

MinerTransaction

  • Extension of TransactionBase
  • First Transaction in each block which contains the Block rewards for the consensus node that produced the Block.
  • Properties:
    • type: 'MinerTransaction'
    • nonce: number

StateTransaction

  • Extension of TransactionBase
  • Properties:
    • type: 'StateTransaction'

Transaction

  • MinerTransaction
  • IssueTransaction
  • ClaimTransaction
  • ContractTransaction
  • StateTransaction
  • InvocationTransaction

TransactionBase

  • Base interface for all Transactions.
  • Properties:
    • version: number
    • hash: Hash256String
      • Hash256 of this Transaction
    • size: number
      • Byte size of this Transaction
    • Attributes: ReadonlyArray<Attribute>
      • Attributes attached to the transaction
    • inputs: ReadonlyArray<Input>
      • Inputs of the Transaction.
    • outputs: ReadonlyArray<Output>
      • Outputs of the Transaction.
    • scripts: ReadonlyArray<Witness>
    • systemFee: BigNumber
      • Fee for invoking a smart contract.
    • networkFee: BigNumber
      • Fee attached to a transaction to increase chances of getting it through quickly.

TransactionOptions

  • Options available when submitting a Transaction or Transfer.
  • Properties (All optional):
    • from: UserAccountID
      • Account to transfer assets from. If this is not provided, the client's current account will be used.
    • attributes: ReadonlyArray<Attribute>
      • Attributes to attach to the transaction.
    • networkFee: BigNumber
      • Fee attached to a transaction to increase chances of getting it through quickly.
    • systemFee: BigNumber
      • Fee for invoking a smart contract.

TransactionReceipt

  • Receipt of a confirmed Transaction which contains data about the confirmation such as the Block index and the index of the Transaction within the block.
  • Properties:
    • blockIndex: number
      • Block index of the Transaction for this receipt.
    • blockHash: Hash256String
      • Block hash of the Transaction for this receipt.
    • transactionIndex: number
      • Transaction index of the Transaction within the Block for this Receipt.

TransactionResult

  • Result of a Transaction
  • Properties:
    • transaction: Transaction
      • Transaction send to the network.
    • confirmed: (options?: GetOptions) => Promise<TransactionReceipt>
      • Function which must be called to confirm the Transaction made it into a Block.


Block Types

Block

  • Extension of Header.
  • Information contained in a Block.
  • Properties:
    • transactions: ReadonlyArray<ConfirmedTransaction>
      • Transactions contained in the Block.

Header

  • Base information about a Block.
  • Properties:
    • version: number
      • NEO blockchain version.
    • hash: Hash256String
      • Block hash
    • previousBlockHash: Hash256String
    • merkleRoot: Hash256String
    • time: number
      • Block time persisted
    • index: number
      • Block index
    • nonce: string
    • nextConsensus: AddressString
      • Next consensus address.
    • script: Witness
    • size: number
      • Size in bytes of the Block


Advanced Types

The rest of the types are contained here. These will likely only be used for more advanced use cases.

AddressAttributeUsage

  • Attribute usage flag indicating the data is a PublicKey 
    'Script'

AddressContractParameter

  • Invocation stack item for an Address
  • See ContractParameter
  • Properties
    • type: 'Address'
    • value: AddressString

ArrayContractParameter

  • Invocation stack item for an Array
  • See ContractParameter
  • Properties
    • type: 'Array'
    • value: ReadonlyArray<ContractParameter>

AssetType

  • An enum of string values of all possible Asset Types 
    'Credit' | 'Duty' | 'Governing' | 'Utility' | 'Currency' | 'Share' | 'Invoice' | 'Token'

AttributeUsage

  • Attribute usage flag indicates the type of the data.
    • BufferAttributeUsage
    • AddressAttributeUsage
    • PublicKeyAttributeUsage
    • Hash256AttributeUsage

BlockFilter

  • Filter user in certain methods which iterate over blocks.
  • Properties (All optional):
    • indexStart: number
    • indexStop: number

BooleanContractParameter

  • Invocation stack item for boolean
  • See ContractParameter
  • Properties
    • type: 'Boolean'
    • value: boolean

BufferAttributeUsage

  • Attribute usage flag indicating the data is an arbitrary Buffer 
    'DescriptionUrl' | 'Description' | 'Remark' | 'Remark1' | 'Remark2' | 'Remark3' | 'Remark4' | 'Remark5' | 'Remark6' | 'Remark7' | 'Remark8' | 'Remark9' | 'Remark10' | 'Remark11' | 'Remark12' | 'Remark13' | 'Remark14' | 'Remark15'

BufferContractParameter

  • Invocation stack item for a Buffer
  • See ContractParameter
  • Properties
    • type: 'Buffer'
    • value: BufferString

ContractParameter

  • ContractParameters are the serialized stack items of an invocation. These are typically the raw results of an invocation, but they may appear in other raw contexts.
    • SignatureContractParameter
    • BooleanContractParameter
    • IntegerContractParameter
    • AddressContractParameter
    • Hash256ContractParameter
    • BufferContractParameter
    • PublicKeyContractParameter
    • StringContractParameter
    • ArrayContractParameter
    • InteropInterfaceContractParameter
    • VoidContractParameter

Hash256AttributeUsage

  • Attribute usage flag indicating the data is a Hash256 
    'ContractHash' | 'Vote' | 'Hash1' | 'Hash2' | 'Hash3' | 'Hash4' | 'Hash5' | 'Hash6' | 'Hash7' | 'Hash8' | 'Hash9' | 'Hash10' | 'Hash11' | 'Hash12' | 'Hash13' | 'Hash14' | 'Hash15'

Hash256ContractParameter

  • Invocation stack item for Hash256
  • See ContractParameter
  • Properties
    • type: 'Hash256'
    • value: Hash256String

IntegerContractParameter

  • Invocation stack item for a BN
  • Note that unlike most of the client APIs, we use a BN instead of a BigNumber here to indicate that this is an integer value.
  • For example, an IntegerContractParameter that represents a NEO value of 10 would be 
    new BN(10_00000000)
  • See ContractParameter
  • Properties
    • type: 'Boolean'
    • value: boolean

InteropInterfaceContractParameter

  • Invocation stack item for anything other than the other valid contract parameters.
  • Examples include the Block builtin. If these builtins remain on the stack after invocation, for example, as a return value, then they will be serialized as this empty interface.
  • See ContractParameter
  • Properties
    • type: 'InteropInterface'

NetworkSettings

  • Settings of the network.
  • Properties:
    • issueGASFee: BigNumber

PrivateNetworkSettings

  • Settings specific to a private network.
  • Properties:
    • secondsPerBlock: number

PublicKeyAttributeUsage

  • Attribute usage flag indicating the data is a PublicKey 
    'ECDH02' | 'ECDH03'

PublicKeyContractParameter

  • Invocation stack item for PublicKey
  • See ContractParameter
  • Properties
    • type: 'PublicKey'
    • value: PublicKeyString

SignatureContractParameter

  • Invocation stack item for Signature
  • See ContractParameter
  • Properties
    • type: 'Signature'
    • value: SignatureString

RawAction

  • Raw action emitted during an invocation.
    • RawNotification
    • RawLog
  • Low-level API for advanced usage only.

RawActionBase

  • Raw action emitted during an invocation.
  • Low-level API for advanced usage only.
  • Properties:
    • version: number
    • blockIndex: number
    • blockHash: Hash256String
    • transactionIndex: number
    • transactionHash: Hash256String
    • index: number
    • globalIndex: BigNumber
    • address: AddressString

RawCallReceipt

  • Raw receipt of an invocation.
  • Low-level API for advanced usage only.
  • Properties:
    • result: RawInvocationResult
    • actions: ReadonlyArray<RawAction>

RawInvocationData

  • Raw data about an invocation.
  • Low-level API for advanced usage only.
  • Properties:
    • asset: Asset
    • contracts: ReadonlyArray<Contract>
    • deletedContractAddresses: ReadonlyArray<AddressString>
    • migratedContractAddresses: ReadonlyArray<[AddressString, AddressString]>
    • result: RawInvocationResult
    • actions: ReadonlyArray<RawAction>

RawInvocationResult

  • Raw result of an invocation.
    • RawInvocationResultError
    • RawInvocationResultSuccess
  • Low-level API for advanced usage only.

RawInvocationResultError

  • Raw result of a failed invocation.
  • Low-level API for advanced usage only.
  • Properties
    • state: 'FAULT'
    • gasConsumed: BigNumber
    • gasCost: BigNumber
    • stack: ReadonlyArray<ContractParameter>
    • message: string

RawInvocationResultSuccess

  • Raw result of a successful invocation.
  • Low-level API for advanced usage only.
  • Properties
    • state: 'HALT'
    • gasConsumed: BigNumber
    • gasCost: BigNumber
    • stack: ReadonlyArray<ContractParameter>

RawInvokeReceipt

  • Extension of TransactionReceipt
  • Raw receipt of an invocation.
  • Low-level API for advanced usage only.
  • Properties:
    • result: RawInvocationResult
    • actions: ReadonlyArray<RawAction>

RawLog

  • Extension of RawActionBase
  • Raw log emitted during an invocation.
  • Low-level API for advanced usage only.
  • Properties:
    • type: 'Log'
    • message: string

RawNotification

  • Extension of RawActionBase
  • Raw notification emitted during an invocation.
  • Low-level API for advanced usage only.
  • Properties:
    • type: 'Notification'
    • args: ReadonlyArray<ContractParameter>

StringContractParameter

  • Invocation stack item for string
  • See ContractParameter
  • Properties
    • type: 'String'
    • value: string

UpdateAccountNameOptions

  • Options for updating a UserAccount name.
  • Properties:
    • id: UserAccountID
      • ID of account to change name.
    • name: string
      • Name to change to.

VoidContractParameter

  • Invocation stack item for void
  • See ContractParameter
  • Properties
    • type: 'Void'

Witness

  • Properties:
    • invocation: BufferString
    • verification: BufferString
