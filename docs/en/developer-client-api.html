Developer Client API

This is the API documentation for the NEO•ONE Developer Client. The Developer Client contains helpful methods for use during development on a private network.

Methods

  • runConsensusNow
  • updateSettings
  • getSettings
  • fastForwardOffset
  • fastForwardToTime
  • reset

Methods Reference

runConsensusNow(): Promise<void>

  • Runs consensus immediately on your private network.
  • Useful for immediately confirming Transactions and not having to wait between Blocks.

updateSettings(Partial<PrivateNetworkSettings>): Promise<void>

  • Updates the PrivateNetworkSettings, like seconds per block, to speed up your network for testing.

getSettings(): Promise<PrivateNetworkSettings>

  • Returns the current PrivateNetworkSettings.

fastForwardOffset(seconds: number): Promise<void>

  • Fast forwards the time of a private network by the given number of seconds. The time of the next Block will be increased by the offset provided in seconds.
  • This can be useful for testing time dependent contracts, such as ICOs.

fastForwardToTime(seconds: number): Promise<void>

  • Fast forwards the time of a private network to the given number of seconds. The time of the next Block will be set to the time provided in seconds.
  • This can be useful for testing time dependent contracts, such as ICOs.

reset(): Promise<void>

  • Resets the private network.
